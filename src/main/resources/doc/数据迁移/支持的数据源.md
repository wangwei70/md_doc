### 支持的数据源

| 源数据                    | 目标数据源                                               | 说明                          |
|------------------------|-----------------------------------------------------|-----------------------------|
| Oracle 11G, 12C,19C    | MySQL<br/>PostgreSQL<br/>AntDB<br/>Oracle<br/>Kafka | 仅到AntDB-T的Oracle兼容模式支持元数据迁移 |
| Mysql5.7<br/> MySQL8.x | AntDB，kafka                                         | 仅到AntDB-T支持元数据迁移            |
 | AntDB-M                | MySQL，kafka                                         | 仅支持数据迁移                     |
| AntDB-T                | ES-7，kafka                                          |                             |
